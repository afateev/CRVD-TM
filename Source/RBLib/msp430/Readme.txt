Данная папка предназначена для хранения реализация модулей общих для семейств и подсемейств контроллеров msp430.
Имена папок формируются исходя из обощения особенностей реализации части функций контроллера.
Например контроллер Uart имеет общую реализацию для всего семейства msp430f5xxx, значит его реализация будет находиться
в папке msp430/f5xxx
Допустим Wdt имеет однотипную реализацию только для семейства msp430f543x, значит его реализация будет находиться
в папке msp430/f5xx/543x

Итак структура папок предполагается следующая:
-msp430
--f5xxx
----543x
--f16xx
--f26xx
и так далее.

Пространства имён в файлах реализаций внутри папок не должны быть вложены, а должны именоваться как конечная точка обощения свойств,
например msp430f5xxx или msp430f543x.
